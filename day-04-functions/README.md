### 🔑 Key Takeaways
1. Functions = Reusability 🔁
write once, use forever. stop copy pasting code 💀
2. Parameters ≠ Arguments 📋
parameter is the placeholder, argument is the actual value you pass in. don't mix them up!
3. Scope is Real 🗺️
local variables die inside the function. global lives everywhere. use global keyword only when you really need to.
**4. *args & kwargs = Flexibility 🎒
when you don't know how many inputs — *args for values, **kwargs for named values. super powerful combo.
5. Always Return, Don't Just Print 🎁
print() just shows it, return actually gives it back so you can USE it. returning > printing inside functions.
6. Lambda = Quick One-Liners ⚡
perfect for short throwaway logic especially inside sorted(), map(), filter(). don't overuse it tho.
7. One Function, One Job 🎯
keep functions small and focused. if it does too many things — split it up!

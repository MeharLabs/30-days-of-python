# 📅 Day 02 — Conditionals

## 🎯 Goal
Learn how to make decisions in Python by using if, elif, and else statements along with comparison, logical, membership, and identity operators — so that program can think, choose, and respond based on different conditions.

## 📚 Topics Covered
Comparisons, Logical, Membership, Identity Operators, Nested Conditionals and Ternary Operators

- Topic 1: Conditionals: if, elif, else
- Topic 2: Comparison Operators: ==, !=, >, <, >=, <=
- Topic 3: Logical Operators: and, or, not
- Topic 4: Membership Operators: in, not in
- Topic 5: Identity Operators: None
- Topic 6: Nested Conditionals: if inside another if
- Topic 7: Ternary/Inline Operator: if else in one line

## 💡 Key Takeaway
✅ if, elif, else are the core building blocks of decision making in Python

✅ Comparison operators (==, !=, >, <, >=, <=) are used to compare values inside conditions

✅ Logical operators (and, or, not) help you combine multiple conditions together

✅ Membership operators (in, not in) check if a value exists in a sequence

✅ Identity operators (is, is not) check if two variables point to the same object

✅ Nested conditionals allow you to check conditions inside other conditions

✅ Ternary if lets you write simple conditions in just one line

✅ elif is better than multiple ifs — it stops after the first match

✅ Clean and simple conditionals make your code more readable and professional

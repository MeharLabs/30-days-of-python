### 🔑 Key Takeaways
🔹 Lists = mutable [ ] — use when data changes

🔹 Tuples = immutable ( ) — use when data is fixed

🔹 Indexing & Slicing works on both

🔹 List methods like .append(), .remove(), .sort(), .pop() modify the list

🔹 Tuple unpacking — x, y = (10, 20) is super powerful 🤯

🔹 List comprehensions = cleaner & shorter code ⚡

🔹 Single-item tuple must have a comma — (5,) not (5) ⚠️

🔹 sorted() returns a new list — .sort() changes the original

🔹 Always pick the right data structure — it makes your code safer & faster 🚀
